## Hardware
- STM32F405RGT6
- Driver de motores TB6612FNG
- 2x Motores coreless 2010 7.4v aliexpress
- DIP Switch 3p SMD
- LiPo 2S ~190 mAh
- 4x Pares emisor receptor IR
- Goma de ruedas Atomic AR001 de 1/28 RC
- Chasis de PCB con soportes de motores y ventilador en PLA
